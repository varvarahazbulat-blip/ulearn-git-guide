# S3. ๏ฟฝ๏ฟฝ๋ซช๏ฟฝ

#### HEAD - โฅช๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๋ซช๏ฟฝ, tag - ไจช๏ฟฝเฎข๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๋ซช๏ฟฝ, branch - ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ HEAD ๏ฟฝ๏ฟฝ๋ซช๏ฟฝ

#### checkout - ๏ฟฝ๏ฟฝเฅฌ๏ฟฝ้ฅญ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ, reset - ๏ฟฝ๏ฟฝเฅฌ๏ฟฝ้ฅญ๏ฟฝ๏ฟฝ ๏ฟฝ ๏ฟฝ๏ฟฝโชฎ๏ฟฝ ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ

#### ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ, ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๋ซช๏ฟฝ, ๏ฟฝ๏ฟฝโ ซ์ญฎ๏ฟฝ - ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ

- `git tag` - ๏ฟฝ๋ขฅ๏ฟฝ๏ฟฝ แฏจแฎช โฅฃ๏ฟฝ๏ฟฝ
- `git tag <tagname>` - แฎง๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ โฅฃ
- `git tag -d <tagname>` - ใค ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ โฅฃ
- `git branch --sort=-committerdate` - ๏ฟฝ๋ขฅ๏ฟฝ๏ฟฝ แฏจแฎช ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝโฎช ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ
- `git branch <branchname>` - แฎง๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ
- `git branch -f <branchname> <to_commit>` - แฎง๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝเฅฌ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ
- `git branch -M <newbranch>` - ๏ฟฝ๏ฟฝเฅจ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ โฅช๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ, ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝแซจ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ ใฆฅ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ
- `git branch -d <branchname>` - ใค ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ
- `git branch -D <branchname>` - ใค ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ, ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝแซจ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ-๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ
- `git checkout -d <commit>` ๏ฟฝ๏ฟฝ๏ฟฝ `git switch --detach <commit>` - ๏ฟฝ๏ฟฝเฅฌ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ HEAD ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ, ๏ฟฝ๏ฟฝ็ฅฌ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ detached HEAD
- `git checkout <branch>` ๏ฟฝ๏ฟฝ๏ฟฝ `git switch <branch>` - ๏ฟฝ๏ฟฝเฅฌ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ HEAD ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ
- `git checkout -b <new_branch>` = `git checkout -b <new_branch> HEAD` ๏ฟฝ๏ฟฝ๏ฟฝ `git switch -c <new_branch>` - แฎง๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ ๏ฟฝ๏ฟฝเฅฉ๏ฟฝ ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ
- `git reset --hard <commit>` - ๏ฟฝ๏ฟฝเฅฌ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ HEAD ๏ฟฝ โฅช๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ <commit>
- `git log --oneline --decorate --graph` - ๏ฟฝ๋ขฅ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝโฎข ๏ฟฝ๏ฟฝ HEAD ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝเฅข๏ฟฝ
- `git log --oneline --decorate --graph --all` - ๏ฟฝ๋ขฅ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝโฎข ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๋ซฎ๏ฟฝ ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝเฅข๏ฟฝ
- `git log --oneline --decorate --graph --all --reflog` - ๏ฟฝ๋ขฅ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝโฎข ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๋ซฎ๏ฟฝ ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ HEAD ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝเฅข๏ฟฝ
- `git reflog show <ref>` - ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝโขจ๏ฟฝ ๏ฟฝ ๏ฟฝ๏ฟฝ๋ซช๏ฟฝ๏ฟฝ
- `git reflog` = `git reflog show HEAD` - ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝโขจ๏ฟฝ ๏ฟฝ HEAD
- `git gc` - ใค ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝใฆญ๏ฟฝ ไ ฉ๏ฟฝ๏ฟฝ ๏ฟฝ ๏ฟฝ๏ฟฝโจฌ๏ฟฝ๏ฟฝ๏ฟฝเฎข๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ เฅฏ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝเจฉ
